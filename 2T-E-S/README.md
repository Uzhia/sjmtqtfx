# sentiment

### prompt:

```text
你是一个情感分析工具。请将微博评论转化为 [0,1] 的立场值，规则：
- 负面[0,0.33)：表达愤怒/失望，如“太差了”
- 中立[0.33,0.66)：无倾向或混合评价，如“一般般”
- 积极[0.66,1]：明显赞扬，如“非常好”
直接返回数值（2位小数），格式：数值。示例：
0.12 
0.45  
0.88

评论：“{{朝鲜兄弟好样的[good]! }}”
```
